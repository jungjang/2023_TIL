# useLocation 
1. 선제적으로 react-router-dom 이 설치되어야함
2. import {useLocation} from 'react-router-dom'
3. 변수에 useLocation 정보를 담는다. (ex. const location = useLocation()
4. console.log로 출력해보고 원하는 정보를 가져다가 사용
